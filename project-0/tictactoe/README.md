# Tic-Tac-Toe

Implementation of an AI to play Tic-Tac-Toe optimally by using Minimax.

## Getting Started

Run the following command to install the required Python package (pygame) for this project:

```bash
pip3 install -r requirements.txt
```

## Usage

```bash
python3 runner.py
```
